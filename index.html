Typography & Interaction, MPSCD24

  1. Project 1: Manuscript
    The first project completed for MPSCD24's T&I class was "Manuscript" - a typesetting project as an introduction to HTML & CSS with a focus on strong typographic fundamentals. The text used is an extract from the Bloomberg article by Paul Ford - "What is Code?".
  2. Project 2: Spread
    This project was a collaboration with classmate Bhroovi Gupta and is an interpretation of "Fuck Content" by Michael Rock, originally published on his website 2x4.
  3. Project 3: Binding
    "Binding" is a collection of texts centered on the theme of translational skills as a designer typeset in the style of a scientific journal as a call back to my previous career in medical research.
